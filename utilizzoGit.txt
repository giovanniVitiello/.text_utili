--------------CREARE UN BRANCH E UNA PULL REQUEST------------

1-git checkout -b <nome branch per convenzione(feature/nome)>
2-adesso ci siamo spostati sul branch feature/nome
3-creare o inserisco i file necessari in questo branch (da locale)
4-git status (controllo i nuovi file inseriti)
5-git add . aggiungo i file in index pronti per essere committati
6-git commit -m "" adesso il file è sulla cartella head locale
7-git push del progetto su remoto (git push origin(nome del mio origin) feature/nome(branch))
8-su remoto adesso avrò una pull request, che dovrà essere accettata
9-dopo che è stata accettata, faccio il merge sul percorso master da remoto
10-adesso faccio il git pull origin master e scarico il file nel master del locale
11-mi ritroverò il file sul percorso master
12-posso in teoria cancellare il percorso branch si prima
13- il comando è : git branch -d feature/nome(branch)
